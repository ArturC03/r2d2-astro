## Basic Syntax

R2D2 uses a clean, modern syntax that should feel familiar to developers coming from other languages.

### Variables and Constants

Variables are declared using `let` and constants with `const`:

```typescript
let mutable = 42;        // Mutable variable
const PI = 3.14159;      // Immutable constant
```

### Basic Types

R2D2 has several built-in types:

```typescript
let integer: i32 = 42;           // 32-bit integer
let float: f64 = 3.14;          // 64-bit floating point
let text: string = "Hello";      // String type
let flag: bool = true;          // Boolean type
let nothing: void = void;       // Void type
```

### Type Inference

R2D2 has smart type inference, so type annotations are often optional:

```typescript
let age = 25;              // Inferred as i32
let name = "R2D2";         // Inferred as string
let active = true;         // Inferred as bool
```

## Functions

Functions are declared using the `fn` keyword:

```typescript
fn add(a: i32, b: i32): i32 {
    return a + b;
}

// Functions with single expressions can be shortened
fn multiply(a: i32, b: i32) = a * b;

// Functions with no return value
fn log(message: string): void {
    // Implementation
}
```

## Control Flow

### If Expressions

```typescript
if condition {
    // code
} else if other_condition {
    // code
} else {
    // code
}

// If expressions can return values
let max = if a > b { a } else { b };
```

### Loops

```typescript
// While loop
while condition {
    // code
}

// For loop over range
for i in 0..10 {
    // code
}

// For loop over array
for item in array {
    // code
}
```

## Modules and Interfaces

### Modules

Modules help organize code into reusable units:

```typescript
module Math {
    fn add(a: i32, b: i32): i32 {
        return a + b;
    }
    
    fn subtract(a: i32, b: i32): i32 {
        return a - b;
    }
}

// Using module functions
let sum = Math.add(5, 3);
```

### Interfaces

Interfaces define contracts that modules can implement:

```typescript
interface Shape {
    fn area(): f64;
    fn perimeter(): f64;
}

module Rectangle implements Shape {
    let width: f64;
    let height: f64;
    
    fn area(): f64 {
        return width * height;
    }
    
    fn perimeter(): f64 {
        return 2 * (width + height);
    }
}
```

## Arrays and Collections

### Arrays

```typescript
// Fixed-size arrays
let numbers: i32[] = [1, 2, 3, 4, 5];

// Array operations
let first = numbers[0];
let length = numbers.length;
```

### Maps

```typescript
let scores = Map<string, i32>();
scores.set("Alice", 100);
scores.set("Bob", 85);

let aliceScore = scores.get("Alice");
```

## Error Handling

R2D2 uses a Result type for error handling:

```typescript
fn divide(a: i32, b: i32): Result<f64, string> {
    if b == 0 {
        return Error("Division by zero");
    }
    return Ok(a as f64 / b as f64);
}

// Using Results
let result = divide(10, 2);
match result {
    Ok(value) => {
        // Use value
    },
    Error(message) => {
        // Handle error
    }
}
```
