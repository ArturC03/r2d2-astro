## System Requirements

Before installing R2D2, ensure your system meets these requirements:

- Node.js 16.0 or higher
- npm 7.0 or higher
- Git (for version control)

## Installation Steps

### 1. Install via npm

The easiest way to install R2D2 is through npm:

```bash
npm install -g r2d2-lang
```

### 2. Verify Installation

Check that R2D2 was installed correctly:

```bash
r2d2 --version
```

### 3. Create a New Project

Create a new R2D2 project:

```bash
r2d2 new my-project
cd my-project
```

This will create a new directory with the following structure:

```
my-project/
├── src/
│   └── main.r2d2
├── package.json
├── r2d2.config.json
└── README.md
```

### 4. Project Setup

Install project dependencies:

```bash
npm install
```

## Troubleshooting

### Common Issues

#### Permission Errors

If you encounter permission errors during global installation:

```bash
sudo npm install -g r2d2-lang
```

#### Path Issues

If the `r2d2` command is not found, ensure your npm global bin directory is in your PATH.
